## 📚 EnjoyHub - Discover Your Next Great Story

Welcome to the EnjoyHub repository! This project is the final result of our Capstone Project (TFG) for the Higher Degree in Web Application Development (DAW).

EnjoyHub is a platform designed for book lovers seeking their next literary adventure. Through an intuitive interface, users can explore and discover books that perfectly match their personal tastes and interests.

## 🚀 Main Features

- Personalized Recommendations: Filtering algorithm based on user genres and preferences.
- Detailed Book Profiles: Comprehensive information including synopses and ratings.
- Responsive Interface: Optimized experience for both desktop and mobile devices.
- User Management: Registration and personalized profiles to save "to-read" lists.

## 🛠️ Tech Stack

The following technologies were used to build this platform:

- Backend: PHP, Laravel  
- BBDD: MySQL  
- Creación de API REST
